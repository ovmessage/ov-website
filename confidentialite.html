Introduction

OV Message est une application de messagerie chiffrée conçue pour protéger vos communications. Cette politique de confidentialité explique comment vos données sont traitées lorsque vous utilisez l'application.

Principe fondamental : OV Message fonctionne entièrement hors ligne, sans connexion à aucun serveur. Les messages sont transmis exclusivement par SMS via votre opérateur mobile. Cette architecture rend la collecte de données techniquement impossible.

Données collectées

OV Message ne collecte aucune donnée personnelle.

Aucune Donnée Transmise

Aucune information personnelle n'est envoyée vers l'extérieur

Aucune Analytique

Aucun système de tracking, d'analytique ou de mesure d'audience

Aucune Publicité

Aucune publicité, aucun traceur, aucun outil marketing

Aucun Profiling

Aucun profil utilisateur n'est créé ou maintenu

Données stockées localement

Les seules données qui existent sont stockées localement sur votre appareil :

  • Vos contacts et leurs informations (nom, numéro de téléphone)
  • Vos conversations (messages chiffrés)
  • Vos clés de chiffrement
  • Vos préférences d'utilisation (langue, paramètres)

Rôle de l'opérateur mobile

Les messages sont transmis par SMS via votre opérateur mobile. Le contenu des messages est chiffré et illisible par l'opérateur.

Cependant, comme pour tout SMS standard, les métadonnées suivantes sont visibles par votre opérateur :

  • Numéro de téléphone de l'expéditeur
  • Numéro de téléphone du destinataire
  • Date et heure d'envoi

Contenu protégé : Même si un SMS est intercepté, le contenu du message reste totalement illisible sans la clé de chiffrement partagée entre vous et votre correspondant.

Stockage des données

Aucune sauvegarde à distance n'est effectuée. Les sauvegardes cloud (iCloud, Google) sont désactivées par l'application.

Stockage Matériel Sécurisé

Vos clés de chiffrement sont stockées dans le coffre-fort matériel de votre appareil (Android Keystore)

Chiffrement Local

Si vous configurez un mot de passe, toutes vos données locales sont chiffrées (AES-256)

Gestion des Clés

Génération et gestion des clés de chiffrement directement sur votre appareil

Intégrité Vérifiée

Chaque message est vérifié automatiquement pour détecter toute altération

Attention : Si vous désinstallez l'application ou perdez votre appareil, vos données sont définitivement perdues. Aucune récupération à distance n'est possible.

Partage de données

OV Message ne partage aucune donnée avec des tiers. Aucune donnée n'est vendue, louée ou transmise à des annonceurs, partenaires commerciaux ou toute autre organisation.

Permissions requises

OV Message demande les permissions suivantes, chacune pour une raison précise :

SMS

Pour envoyer et recevoir des messages chiffrés. C'est la fonctionnalité principale de l'application.

Contacts

Uniquement si vous choisissez d'importer des contacts depuis votre répertoire. Cette opération est locale.

Notifications

Pour vous informer des nouveaux messages entrants.

Caméra

Pour scanner les QR codes de partage de clés. Aucune photo n'est stockée ou transmise.

Stockage

Pour l'export de clés de chiffrement sur un support physique. Aucun fichier n'est envoyé à l'extérieur.

Permissions à la demande : Toutes les permissions sont demandées au moment où elles sont nécessaires. Vous pouvez refuser toute permission.

Suppression des données

Vous gardez un contrôle total sur la suppression de vos données :

Messages Éphémères

Suppression automatique des messages après un délai choisi (5 min à 1 mois)

Suppression Manuelle

Vous pouvez supprimer manuellement des conversations à tout moment

Mode Panic

Destruction instantanée et invisible de toutes vos données en cas de menace

Désinstallation

La désinstallation de l'application supprime toutes vos données

Sécurité

Chiffrement des messages

Vos messages sont chiffrés avant d'être envoyés. Seuls vous et votre correspondant, qui partagez la même clé, pouvez lire les messages échangés. Même si les SMS sont interceptés par un tiers, le contenu reste illisible sans la clé.

Messages Uniques

Chaque message produit un résultat différent, même avec le même texte

Anti-falsification

Chaque message est signé pour détecter toute modification

Anti-rejeu

Un message ne peut pas être renvoyé frauduleusement par un tiers

Stockage Matériel

Vos clés sont protégées par le coffre-fort matériel de votre appareil

Chiffrement conditionnel du stockage : Si vous configurez un mot de passe de connexion, vos données locales (contacts, messages, préférences) sont chiffrées sur votre appareil. Sans mot de passe, les données locales ne sont pas chiffrées.

Responsabilité de l'utilisateur

La sécurité de vos communications dépend de la bonne gestion de vos clés de chiffrement. L'échange de clés entre correspondants se fait sous votre responsabilité.

Recommandation : Nous recommandons d'échanger vos clés de chiffrement en personne, via QR code, pour garantir qu'aucun tiers ne puisse intercepter la clé.

Vous êtes également responsable de :

  • La sauvegarde de vos clés de chiffrement (en cas de perte de l'appareil)
  • Le choix d'un mot de passe fort pour protéger le stockage local
  • La rotation régulière de vos clés de chiffrement

Enfants

OV Message n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons sciemment aucune donnée relative aux enfants.
