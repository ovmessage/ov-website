1. Acceptation des conditions

En utilisant OV Message, vous acceptez les présentes conditions d'utilisation. Si vous n'acceptez pas ces conditions, veuillez ne pas utiliser l'application.

Acceptation : L'utilisation de l'application vaut acceptation pleine et entière des présentes conditions.

2. Description du service

Les tarifs SMS standard de votre opérateur mobile s'appliquent pour l'envoi de messages.

3. Utilisation autorisée

L'application est destinée à :

  • Communiquer de manière privée et sécurisée avec vos correspondants
  • Chiffrer vos messages personnels avant envoi par SMS
  • Gérer, partager et renouveler vos clés de chiffrement

Législation locale : Certains pays peuvent restreindre l'utilisation de technologies de chiffrement. Il est de votre responsabilité de vérifier la légalité de l'utilisation dans votre juridiction.

4. Utilisations interdites

Il est strictement interdit d'utiliser OV Message pour :

  • Activités illégales : toute activité contraire à la loi

  • Spam : envoyer des messages non sollicités en masse
  • Contournement de sécurité : tenter de contourner les mécanismes de protection de l'application
  • Distribution non autorisée : distribuer ou modifier le code source de l'application

5. Responsabilité de l'utilisateur

Vous êtes responsable de :

  • Le contenu des messages que vous envoyez via l'application
  • La protection de votre appareil contre les accès non autorisés

Perte de clés et Mode Panic : La perte de vos clés de chiffrement ou l'activation du Mode Panic entraînent la perte définitive de vos données. OV Message ne peut pas récupérer vos clés ni vos messages.

Aucune technologie infaillible : Bien que OV Message utilise des technologies de chiffrement avancées, aucune technologie ne peut garantir une sécurité absolue contre toutes les menaces.
